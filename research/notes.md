# Заметки по исследованию ЕАСУЗ

## ✅ ЭТАП 2 ЗАВЕРШЕН!

### API информация
- **URL**: https://easuz.mosreg.ru/api/v1-web/Purchase/GetPurchasePage
- **Метод**: POST
- **Content-Type**: application/json
- **Всего объявлений**: ~3095
- **Пагинация**: 10 объявлений на страницу

### Структура запроса
```json
{
  "filter": {
    "purchaseStageState": [1000004]  // Торги объявлены
  },
  "page": 1,
  "take": 10
}
```

### Данные в ответе
- ✅ ID объявления
- ✅ Название
- ✅ Цена (startPrice)
- ✅ Площадь (totalSquare)
- ✅ Адрес (addressDescription)
- ✅ Координаты (latitude, longitude)
- ✅ Фото (photos[])
- ✅ Срок аренды (rightTermUseYear)
- ✅ Количество просмотров
